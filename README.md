# -PVR.CCZ-Plist-Batch-Extractor
Este é um script de automação em Python desenvolvido para processar em lote (batch) arquivos de atlas de sprites comprimidos em formato .pvr.ccz acompanhados de seus respectivos arquivos .plist.  O diferencial desta ferramenta é a integração com o TexturePacker CLI, permitindo a extração de texturas mesmo quando utilizam compressão de hardware.
